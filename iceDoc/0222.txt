串接機制：
目前「屬性組」與「料號建立」頁面透過 localStorage["JY_ATTR_V3_FINAL"] 傳遞 JSON 資料。若資料不存在，頁面會自動 fallback 回測試資料，細節見文檔 3.2 節。

 後續工作 (Mock 轉 API)：
目前均為純前端模擬，需請你協助開發以下 API：

流水號：轉為後端取號 (POST /api/serial/next)。
SAP：串接實體 BAPI/RFC。
資料清單：屬性組、工廠別、BOM 清單改為 API 獲取。
帳號管理：需完整 CRUD 功能。
